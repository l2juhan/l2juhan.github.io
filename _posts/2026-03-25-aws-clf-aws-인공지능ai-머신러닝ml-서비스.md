## 1: AI 서비스 (API 기반 서비스)

- 머신러닝 전문 지식이 없어도, 간단한 API 호출만으로 강력한 AI 기능을 내 애플리케이션에 바로 추가할 수 있습니다. 

### 대화형 AI

- Amazon Lex
- **핵심 역할:** 음성 및 텍스트 기반 **챗봇** 제작 ("챗봇의 뇌와 입")
- **설명:** Amazon Alexa를 만드는 데 사용된 것과 동일한 기술을 사용하여, 사용자의 말을 이해하고 자연스러운 대화를 이끌어가는 챗봇을 쉽게 만들 수 있습니다.

### 음성/텍스트 변환

- Amazon Polly
- **핵심 역할:** 텍스트를 사람처럼 자연스러운 **음성으로 변환** (TTS, Text-to-Speech) ("AI 성우")
- **설명:** 블로그 글이나 뉴스를 오디오로 들려주거나, 애플리케이션의 음성 안내를 만드는 데 사용됩니다.
- **Amazon Transcribe**
- **핵심 역할:** 음성을 **텍스트로 변환** (STT, Speech-to-Text) ("자동 받아쓰기 전문가")
- **설명:** 고객센터 통화 내용을 자동으로 기록하거나, 동영상에 자막을 생성하는 데 사용됩니다.

### 문서 및 데이터 추출

- Amazon Textract
- **핵심 역할:** 스캔한 문서나 이미지(PDF, JPG)에서 **텍스트와 표 데이터를 추출** ("OCR 전문가")
- **설명:** 단순 텍스트뿐만 아니라 표의 행과 열 구조까지 인식하여 추출할 수 있습니다. 영수증, 계약서 처리 자동화에 유용합니다.

### 지능형 검색

- Amazon Kendra
- **핵심 역할:** 기업용 **지능형 검색 엔진** ("AI 사서")
- **설명:** 회사 내부 문서나 FAQ를 학습하여, 사용자가 자연어로 질문했을 때 단순 링크가 아닌 **정확한 답변**을 찾아줍니다.

### 이미지 및 비디오 분석

- Amazon Rekognition
- **핵심 역할:** 이미지와 비디오를 분석하여 **객체, 사람, 텍스트, 장면 등을 식별** ("AI의 눈")
- **설명:** 사진 속 유명인사를 인식하거나, 부적절한 콘텐츠를 감지하고, 영상 속 인물의 동선을 추적하는 등의 작업이 가능합니다.

### 번역 및 언어 처리

- Amazon Translate
- **핵심 역할:** 텍스트를 다양한 언어로 자동 **번역**
- **설명:** 실시간으로 웹사이트 콘텐츠나 사용자 채팅을 번역해 줍니다.
- Amazon Comprehend
- **핵심 역할:** 텍스트에서 **핵심 구문, 감정(긍정/부정), 주제 등을 분석** ("문맥 파악 전문가")
- **설명:** 고객 리뷰가 긍정적인지 부정적인지 자동으로 분석하거나, 문서의 핵심 주제를 파악하는 데 사용됩니다.

### 추천 및 예측

- Amazon Personalize
- **핵심 역할:** 사용자에게 맞는 **개인화된 상품, 콘텐츠를 추천**해주는 엔진 구축
- **설명:** Amazon.com에서 사용하는 것과 동일한 기술로, "이 상품을 본 고객이 함께 본 상품"과 같은 추천 시스템을 쉽게 만들 수 있습니다.
- Amazon Forecast
- **핵심 역할:** 시계열 데이터를 기반으로 미래 수요를 **예측** (예: 재고, 매출 예측)
- **설명:** 과거 판매 데이터를 기반으로 "다음 분기에 이 제품이 몇 개나 팔릴까?"를 예측하여 재고 관리를 최적화합니다.

## 2: ML 플랫폼

- 데이터 과학자나 개발자가 자신만의 머신러닝 모델을 쉽게 **구축, 학습, 배포**할 수 있도록 도와주는 종합 플랫폼입니다. 
- Amazon SageMaker
- **핵심 역할:** 머신러닝 모델 개발의 전 과정을 위한 **통합 개발 환경(IDE)**
- **설명:** 데이터 준비부터 모델 학습, 튜닝, 배포, 모니터링까지 모든 단계를 하나의 플랫폼에서 수행할 수 있게 해주는 **ML 전문가들의 필수 도구**

## ML 프레임워크 및 인프라

- 머신러닝 전문가들이 TensorFlow, PyTorch 같은 딥러닝 프레임워크를 직접 사용하여 모델을 개발할 때, 필요한 고성능 컴퓨팅 인프라를 제공합니다.
- 딥러닝 AMI 및 컨테이너
- **설명:** EC2 인스턴스에 TensorFlow, PyTorch 등 인기 있는 ML 프레임워크와 드라이버가 미리 설치되어 있어, 복잡한 설정 없이 바로 딥러닝 개발을 시작할 수 있도록 제공하는 이미지입니다.
- 고성능 컴퓨팅 인스턴스
- **설명:** GPU(p, g 시리즈)나 AWS가 자체 제작한 AI 칩(Inferentia, Trainium)이 탑재된 EC2 인스턴스로, 대규모 모델 학습 및 추론에 필요한 강력한 컴퓨팅 파워를 제공합니다.
